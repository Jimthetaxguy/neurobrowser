## What was deferred to v0.2 (per the plan)

1. **Native function-calling** — `tools::ToolCall::parse_native` exists;
   the ReActAgent prompt does not yet ask providers to emit structured
   JSON. v0.2 update: change `build_context` tool spec to require
   JSON when the provider supports it.
2. **React Workers sidebar (E5)** — types and IPC endpoints are in
   place; the UI is not. v0.2 add a panel to `App.jsx`.
3. **Headless worker fan-out (E6)** — daemon currently exposes
   policy/snapshot methods. v0.2 add `worker.spawn` + `worker.list`.
4. **Real-LLM end-to-end integration test** — unit tests use a stub
   provider; a real-provider test should be budget-capped via Infisical.
